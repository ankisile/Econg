## K-해커톤 10회 Team Oasis Econg
### 에콩

- 친환경 제품 크라우드 펀딩 앱

- 제로웨이스트 운동의 시장 가능성을 연결시킨 친환경 제품을 제공해주는 앱

- 크라우드 펀딩: 구매하고 싶은 제품을 크라우드 펀딩을 이용하여 펀딩

- 친환경 제품 및 크리에이터 리스트: 친환경 네품 및 크리에이터 리스트를 제공하여 사용자가 접근하기 용이하게 만듦

- 결제기능은 카카오 페이로 구현
